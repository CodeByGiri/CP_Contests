## Problem: 3,2,1 Go!!

### Idea
We need to print numbers from `N` down to `1` in decreasing order.

The only twist:
→ Output should be **comma-separated**.

### Approach
- Start from `N` and go till `1`
- Print each number
- Add comma after every number except the last one

### Complexity
Time: O(N)  
Space: O(1)

### Code

```cpp
#include <iostream>
using namespace std;

int main(){
  int n;
  cin >> n;
  
  while(n > 0){ 
    cout << n;
    if(n != 1) cout << ",";
    n--;
  }
}
```
